Most recent work

PuzzlePeace

PuzzlePeace is a collaborative project designed to enhance the emotional well-being of individuals with autism. Using React Native (JavaScript), Expo, and CSS, the platform provides a refined and responsive interface. Firebase with NoSQL supports real-time data management, enabling seamless journaling of daily experiences, emotions, and behaviors. The platform offers personalized emotional analysis for caregivers and teachers, delivering tailored insights to support individuals with autism effectively.

Demo

CensorMate

CensorMate is a Chrome extension developed using HTML, CSS, and JavaScript that allows users to block and replace specific words on any webpage. Users can customize both the words to block and their replacements, with changes applied dynamically using the MutationObserver Web API to monitor and update text as the page content changes. The extension also uses the Chrome Storage API to securely save and manage user settings. With password protection to prevent unauthorized access, CensorMate is ideal for use around children, ensuring that the blocked words list and its settings remain secure and unaltered.

File Converter

Developed with Python, this application is a versatile file converter that allows users to convert various file types, including images, text files, CSVs, PDFs, and Word Documents into different formats. It supports converting images between common formats like JPG and PNG, transforming text files into PDFs, converting CSV files into Excel spreadsheets and SQL insert statements, and more. The program guides users through the conversion process with simple text-based input, making it a powerful tool for handling diverse file conversions efficiently.
